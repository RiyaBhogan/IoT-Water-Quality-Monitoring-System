# IoT Water Quality Monitoring System

## 📌 Project Overview

This project monitors water quality using sensors and displays results on an LCD. A buzzer alerts when water quality becomes poor.

## 🧰 Components Used

* Arduino Uno
* LCD (16x2)
* Temperature Sensor (TMP)
* LDR / Turbidity (simulated)
* Potentiometer
* Buzzer
* Resistors & Breadboard

## ⚙️ Working

* Sensors collect data (temperature, turbidity/light)
* Arduino processes values
* LCD displays readings
* If values cross threshold → buzzer turns ON
